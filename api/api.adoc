Projects
~~~~~~~~

[cols="3*", options="header"]
|===
| URL
| Method
| Functionality

| /api/v1/projects
| GET
| List projects

| /api/v1/projects
| POST
| Create project

| /api/v1/projects/<projectId>
| GET
| Get project

| /api/v1/projects/<projectId>
| PUT
| Modify project

| /projects/<projectId>
| PATCH
| Modify partialy a project

| /api/v1/projects/<projectId>
| DELETE
| Delete a project
|===
